1. Introduction

NeuroLaunch provides a powerful RESTful API to interact with all major AI features of your SaaS environment. With secure endpoints, JSON responses, and scalable design, this API is perfect for developers looking to integrate AI magic into their apps.

2. Authentication

Use your API key from the dashboard to authenticate all your requests. Always keep your key safe.

GET /v1/user
Headers:
  Authorization: Bearer YOUR_API_KEY

3. Endpoints Overview

  • /v1/generate - Generate AI-powered content.
  • /v1/sentiment - Analyze sentiment in user text.
  • /v1/chat - Initiate a chat session with the AI.

4. Example: Generate Text

POST /v1/generate
Headers:
  Authorization: Bearer YOUR_API_KEY
  Content-Type: application/json

Body:
{
  "prompt": "Write a tagline for a new fitness app",
  "max_tokens": 40
}

Response:

{
  "result": "Get fit, stay strong, feel great‚Äîanytime, anywhere."
}

5. Rate Limiting

Each plan has a specific rate limit. Free users can make 100 requests per day. Pro users enjoy 10,000 requests/day. View your usage on the dashboard.

6. Error Codes

  • 401 Unauthorized: Invalid or missing API key.
  • 429 Too Many Requests: Rate limit exceeded.
  • 500 Internal Server Error: Something went wrong.

7. Webhook Support

We support webhooks for real-time updates. You can register a webhook endpoint in your settings.

8. SDKs & Libraries

Coming soon: JavaScript, Python, and PHP SDKs. For now, you can use standard fetch/Axios to make requests.
