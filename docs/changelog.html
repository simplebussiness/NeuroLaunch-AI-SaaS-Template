Changelog

Stay up to date with all the latest features, improvements, and bug fixes implemented in NeuroLaunch. Our changelog records every update to help you understand what’s new and how the platform evolves.

Version 2.5.0 - June 2025

  • 🚀 Added multi-language support for the API responses.
  • ✨ New integration with Microsoft Teams for collaboration.
  • 🛠️ Improved webhook security with OAuth2 authentication.
  • 🐛 Fixed latency issues in the analytics dashboard.
  • 📈 Enhanced model training logs and visualization.

Version 2.4.0 - April 2025

  • 🆕 Introduced AI-powered content summarization tools.
  • ⚙️ Refined user roles and permissions in dashboard.
  • 🐞 Resolved bugs related to report exports.
  • 🔗 Added Zapier workflows for automatic lead capture.
  • 📚 Updated documentation for API endpoints.

Version 2.3.1 - March 2025

  • 🔒 Patched security vulnerabilities reported by community.
  • 📊 Improved real-time data sync across devices.
  • 🛠️ Fixed integration issues with Google Sheets add-on.
  • 📞 Added webhook retry logic for failed calls.

Version 2.3.0 - February 2025

  • ✨ Launched beta version of voice-command API interface.
  • ⚙️ Enhanced AI model customization settings.
  • 🐛 Fixed bugs in Slack integration workflow.
  • 📈 Added usage stats and quota limits for all plans.

Version 2.2.0 - January 2025

  • 🆕 Released new UI for the NeuroLaunch dashboard.
  • 🔧 Added ability to export data in CSV and JSON formats.
  • 🐞 Fixed multiple minor bugs reported by users.
  • 📢 Improved notification system for integration events.

Initial Release - December 2024

  • 🎉 Launched NeuroLaunch platform with core AI features.
  • 📊 Basic analytics and reporting dashboards.
  • 🔐 Secure user authentication and role management.
  • 📚 Comprehensive documentation and tutorials.
